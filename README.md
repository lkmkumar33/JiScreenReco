## Requirements

- Windows 10/11
- .NET Framework 4.7.2
- FFmpeg (automatically detected or can be downloaded)

## Installation

1. Download the latest release
2. Extract to your preferred location
3. Run `JiScreenReco.exe`

## Usage

- **F9** - Start recording
- **F10** - Stop recording  
- **F11** - Extract frames from video
- **Right-click** - Change recording area
